# Satellite Image Classification using Deep Learning

## Overview

Satellite Image Classification is a deep learning-based application that classifies satellite images into different land-cover categories. The project uses image preprocessing techniques and Convolutional Neural Networks (CNNs) to analyze and predict image classes.

## Features

* Satellite image preprocessing
* CNN-based image classification
* User-friendly graphical interface
* Fast image prediction
* Sample image testing support

## Technologies Used

* Python
* TensorFlow / Keras
* OpenCV
* NumPy
* Pandas
* Scikit-learn
* Matplotlib
* Tkinter

## Project Structure

* Dataset/ – Training dataset
* model/ – Trained model files
* sampleImages/ – Sample images for testing
* SatelliteImageClassification.py – Main application

## How to Run

1. Install the required dependencies:
   pip install -r requirements.txt

2. Run the application:
   python SatelliteImageClassification.py
